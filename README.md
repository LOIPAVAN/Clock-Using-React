# React Clock Application

>A simple digital clock built with React and Vite.

## Features
- Displays the current time, updating every second
- Responsive and clean UI
- Built with React functional components

## Getting Started

### Prerequisites
- Node.js (v16 or higher recommended)
- npm or yarn

### Installation
1. Clone the repository:
	```bash
	git clone <your-repo-url>
	cd Clock
	```
2. Install dependencies:
	```bash
	npm install
	# or
	yarn install
	```

### Running the App
Start the development server:
```bash
npm run dev
# or
yarn dev
```
Open your browser and go to [http://localhost:5173](http://localhost:5173) (or the port shown in your terminal).
